Mouse Controls:
- `Command + Left`: Rotate camera
- `Right`: Look around
- `Middle`: Pan
- `Scroll`: Zoom in/out

Keyboard Controls:
- `Tab`: Select next in-scene object
- `F`: Focus on object
- `Q`: No action
- `W`: Translate
- `E`: Rotate
- `R`: Scale
